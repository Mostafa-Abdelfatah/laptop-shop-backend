User APIs:-

### **1) Get All Users (Search)**

**Endpoint:**
```
GET /user/getAllUsers
```

**Description:**
Retrieve a paginated list of users with optional filters and sorting by creation date.

**Query Parameters:**

| Parameter | Type    | Required | Default | Description                                                 |
| --------- | ------- | -------- | ------- | ----------------------------------------------------------- |
| `page`    | Integer | No       | `1`     | The page number to retrieve. Starts from 1.                 |
| `limit`   | Integer | No       | `10`    | The number of users to retrieve per page.                   |
| `Fname`   | String  | No       | `-`     | Filter users by their first name. (Case-insensitive search) |
| `Lname`   | String  | No       | `-`     | Filter users by their last name. (Case-insensitive search)  |
| `email`   | String  | No       | `-`     | Filter users by their email. (Case-insensitive search)      |
| `sort`    | String  | No       | `desc`  | Sort the results by creation date. Use `asc` or `desc`.     |

**Responses:**

#### **Success (200):**

```json
{
  "success": true,
  "totalUsers": 25,
  "currentPage": 1,
  "totalPages": 3,
  "data": [
    {
      "_id": "63a1b2c34f8e5c0024c6ef12",
      "Fname": "John",
      "Lname": "Doe",
      "email": "john.doe@example.com",
      "Date_created": "2024-12-19T10:10:30.511Z"
    },
    {
      "_id": "63a1b2c34f8e5c0024c6ef13",
      "Fname": "Jane",
      "Lname": "Smith",
      "email": "jane.smith@example.com",
      "Date_created": "2024-12-18T09:15:30.511Z"
    }
  ]
}
```

#### **Error Responses:**

| Status Code | Message                                    | Description                                |
| ----------- | ------------------------------------------ | ------------------------------------------ |
| `400`       | "Invalid sort value. Use 'asc' or 'desc'." | The provided `sort` parameter is invalid.  |
| `400`       | "Page must be a positive integer."         | The provided `page` parameter is invalid.  |
| `400`       | "Limit must be a positive integer."        | The provided `limit` parameter is invalid. |
| `404`       | "No users found"                           | No users match the filter criteria.        |
| `500`       | "Internal Server Error"                    | A server-side error occurred.              |

**Example Requests and Responses:**

- **Request:**
  ```
  GET /user/getAllUsers?page=2&limit=5&sort=asc
  ```
  **Response:**
  ```json
  {
    "success": true,
    "totalUsers": 100,
    "currentPage": 2,
    "totalPages": 20,
    "data": [
      {
        "_id": "63a1b2c34f8e5c0024c6ef15",
        "Fname": "Alice",
        "Lname": "Wonderland",
        "email": "alice@example.com",
        "Date_created": "2024-12-19T10:20:30.511Z"
      }
    ]
  }
  ```

**Notes:**
- If no query parameters are provided, the endpoint returns the first page of users with 10 users per page, sorted by descending creation date.
- Optional query parameters (e.g., `Fname`, `Lname`, `email`) allow flexible filtering.

### **1) Get All Laptops**

**Endpoint:**
```
GET /laptop/getAllLaptops
```

**Description:**
Retrieve all laptops with optional filtering, sorting, and pagination.

**Query Parameters:**

| Parameter | Type    | Required | Default | Description                                  |
| Parameter     | Type    | Description                                                                 | Required | Example              |
|---------------|---------|-----------------------------------------------------------------------------|----------|----------------------|
| stock         | string  | Filter laptops by stock status: "in" (in-stock) or "out" (out-of-stock).   | No       | in                   |
| brandName     | string  | Filter laptops by brand name (case-insensitive, partial match).             | No       | Dell                 |
| categoryName  | string  | Filter laptops by category name (case-insensitive, partial match).          | No       | Gaming               |
| laptopName    | string  | Filter laptops by name (case-insensitive, partial match).                   | No       | MacBook              |
| cpu           | string  | Filter laptops by CPU name (case-insensitive, partial match).               | No       | Intel Core i7        |
| gpu           | string  | Filter laptops by GPU name (case-insensitive, partial match).               | No       | NVIDIA RTX 3060      |
| ram           | string  | Filter laptops by RAM size (case-insensitive, partial match).               | No       | 16GB                 |
| storage       | string  | Filter laptops by storage type (case-insensitive, partial match).           | No       | 512GB SSD            |
| rate          | number  | Filter laptops by exact rating (0-5).                                       | No       | 4.5                  |
| sort          | string  | Sort laptops by price: "asc" (ascending) or "desc" (descending).            | No       | asc                  |
| page          | number  | Page number for pagination.                                                 | No       | 1                    |
| limit         | number  | Number of items per page for pagination.

Response Fields:

| Field         | Type   | Description                                                                 |
|---------------|--------|-----------------------------------------------------------------------------|
| totalLaptops  | number | The total number of laptops matching the query.                            |
| totalPages    | number | Total number of pages.                                                     |
| currentPage   | number | Current page number.                                                       |
| laptops       | array  | Array of laptop objects containing detailed information.                   |

Laptop Object Fields:

| Field         | Type    | Description                                                               |
|---------------|---------|---------------------------------------------------------------------------|
| _id           | string  | Unique identifier for the laptop.                                        |
| name          | string  | Name of the laptop.                                                      |
| description   | string  | Description of the laptop.                                               |
| photo         | string  | URL of the laptop's image.                                               |
| stock         | number  | Quantity of laptops in stock.                                            |
| price         | number  | Price of the laptop.                                                     |
| category_id   | string  | Identifier for the laptop's category.                                    |
| brand_id      | string  | Identifier for the laptop's brand.                                       |
| rate          | number  | Average rating of the laptop (0-5).                                      |
| storage       | string  | Storage details (e.g., 512GB SSD).                                       |
| ram           | string  | RAM details (e.g., 16GB).                                                |
| gpu           | string  | GPU details (e.g., NVIDIA RTX 3060).                                     |
| cpu           | string  | CPU details (e.g., Intel Core i7).                                       |
| Date_created  | string  | The date the laptop was added.                                           |
| Date_updated  | string  | The date the laptop was last updated.                                    |

Shape Examples:
GET /laptop/getAllLaptops?categoryName=Gaming&sort=desc
GET /laptop/getAllLaptops?laptopName=MacBook
